# Spielkonzept – IFeelDump Quiz

## Kernidee
Ein moderiertes, lokales Quizspiel für 1–4 Spieler an einem Computer.

## Ablauf
1. Startseite
2. Button: "Lokales Spiel starten"
3. Setup-Fenster im Spiel
4. Moderator konfiguriert:
   - Anzahl der Spieler
   - Spielernamen
   - Anzahl Fragen
   - Kategorie
   - Zeit pro Frage
5. Spielstart
6. Per visueller Animation wird ein Startspieler bestimmt
7. Spieler antworten nacheinander verdeckt
8. Moderator kann vor der Auflösung noch etwas erzählen
9. Moderator klickt auf "Antwort auflösen"
10. Richtige Antwort und richtige Spieler werden angezeigt
11. Nächste Frage
12. Nach letzter Frage: "Zur Statistik"
13. Ergebnisse speichern
14. Historie anzeigen

## Warum kein Simultanmodus als Standard?
Weil 1 Maus + 1 Tastatur für 4 lokale Spieler unpraktisch ist.
Der moderierte Reihum-Modus ist für das gewünschte Setup deutlich besser geeignet.

## Steuerung
- Maus
- Tastatur 1–4
- alternativ A–D

## Moderatorrolle
Der Moderator ist bewusst Teil des Spiels:
- startet Setup
- kann während der Frage sprechen
- löst die Antwort manuell auf
- startet die nächste Frage

## Spielende
Anstelle von "Nächste Frage" erscheint nach der letzten Frage:
- Button: "Zur Statistik"

## Statistik
Je Spieler:
- Punkte
- richtige Antworten
- falsche Antworten
- Trefferquote

## Historie
Eine Übersicht, in der sichtbar ist:
- wann ein Spiel war
- welche Kategorie gespielt wurde
- wie viele Fragen es gab
- welche Spieler teilgenommen haben
- wie viele Punkte jeder hatte

## Löschfunktionen
- einzelnes Spiel löschen
- gesamte Historie löschen
